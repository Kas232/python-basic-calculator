# 🧮 Python Basic Calculator

A simple command-line calculator built with **Python**. This beginner-friendly project performs basic arithmetic operations using user input and Python's `match-case` statement.

## 🚀 Live Demo

🌐 **Live Website:** https://kas232.github.io/python-basic-calculator/

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* ⚠️ Handles invalid operators
* 🛡️ Prevents division by zero
* 💻 Simple command-line interface

## 🛠️ Built With

* Python 3

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/kas232/python-basic-calculator.git
   ```

2. Open the project folder.

3. Run the program:

   ```bash
   python calculator.py
   ```
